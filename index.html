Technical Specifications

SubtitleVision is an iOS app written in Swift primarily using SwiftUI for layout and UI. Our backend is writtent in Python and uses open source forced alignment models combined with PyTorch. Summary is powered by the ChatGPT API. For further information view our architecture documentation below.
